## Farbpalette für Wix

| Farbe | HEX Code | Verwendung |
|-------|----------|------------|
| Primary Orange | `#F59E0B` | CTAs, Akzente, Icons |
| Dark Orange | `#EA580C` | Hover-States |
| Light Orange BG | `#FEF3C7` | Hintergründe |
| Dark Gray | `#111827` | Footer, Text |
| Medium Gray | `#4B5563` | Subtexte |
| Light Gray | `#F9FAFB` | Section-Hintergründe |
| White | `#FFFFFF` | Haupthintergrund |

## Schriftarten Empfehlung

- **Headlines:** Inter Bold oder Poppins Bold
- **Body:** Inter Regular oder Open Sans
- **Alternative:** System fonts (Apple system, Segoe UI)

## Wix Implementation Tips

1. **Sections erstellen:** Jede Section als Strip/Section in Wix anlegen
2. **Farben:** Global Colors in Wix definieren mit obigen HEX-Codes
3. **Buttons:** Pill-shape (rounded-full) mit amber/orange Farbe
4. **Abstände:** Grosszügige Padding (80px top/bottom für Sections)
5. **Mobile:** Immer Mobile-First testen
